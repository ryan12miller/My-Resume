Sturcture of a Table

table - table start

tr - table row start

td - table data (include as many td's as there are columns)

/tr - table row end

/table - table end



Form Notes


Types of Inputs


Checkbox - A check box allowing single values to be selected

Radio - A radio button, allowing a single value to be selected out of multiple choices with the same name value.





Date - A control for entering a date (year, month, and day, with no time). Opens a date picker or numeric wheels for year, month, day when active in supporting browsers.

File - A control that lets the user select a file. Use the accept attribute to define the types of files that the control can select.
